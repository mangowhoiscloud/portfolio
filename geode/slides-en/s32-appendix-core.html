B2. Hook Event Emission 분석 (S14 보강)

High-frequency (매 파이프라인 발화)

PIPELINE_START/END

NODE_ENTER/EXIT

LLM_CALL_START/END

TURN_COMPLETE

SESSION_START/END

RunLog에 전수 기록. 관측의 기반이 됩니다.

Medium-frequency (조건부 발화)

VERIFICATION_PASS/FAIL

SCORING_COMPLETE

TOOL_APPROVAL_*

CONTEXT_CRITICAL

분석/검증 세션에서만 발화. L2 React 자동 반응 트리거.

Rare but critical (장애 시에만 발화)

PIPELINE_ERROR

PIPELINE_TIMEOUT

FALLBACK_CROSS_PROVIDER

TOOL_RECOVERY_*

SUBAGENT_FAILED

발화 빈도가 낮지만 삭제하면 장애 시 원인 추적이 불가능해집니다.

왜 42개 전부 필요한가: v0.35.1에서 발화 0인 6개를 삭제했습니다. 남은 42개는 모두 1회 이상 실제 발화 이력이 있거나 장애 시 필수 관측 경로입니다.
