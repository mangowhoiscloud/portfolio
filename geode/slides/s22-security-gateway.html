Security + Gateway: Swiss Cheese 방어

Bash 3-Layer + Secret Redaction 8패턴 + ChannelBinding 세션 격리로 자율 실행 경계를 제어합니다.

Layer 1 — 9 차단 패턴

  • rm -rf / · sudo · >/etc/

  • curl|sh · wget|sh · mkfs.

  • dd if→/dev/ · chmod 777 /

  • fork bomb :(){:|:&}

Layer 2 — Resource Limits

  • CPU 30s 하드 캡

  • 파일 크기 50MB 상한

  • 자식 프로세스 64개 상한

  • preexec_fn setrlimit 적용

Layer 3 — HITL 4-Tier

  • SAFE — 30 prefix 자동 승인

  • WRITE — 개별 승인 필요

  • EXPENSIVE — 비용 일괄 승인

  • DANGEROUS — 순차 승인

Trade-off — safe-first 허용 시 파이프 주입 통과

cat file.txt | bash는 cat prefix로 SAFE 분류 → 파이프 주입이 통과됩니다. Layer 1의 curl/wget 차단이 원격 코드 실행을 방어합니다.

PolicyChain 설계 원칙 — AND-logic

도구는 모든 정책을 통과해야 실행됩니다. Org 정책(priority=5)이 최우선. 6개 레이어 중 하나라도 거부하면 차단. Swiss Cheese: 단일 실패를 다음 레이어가 차단합니다.

Dual Trust Model

GEODE는 두 가지 신뢰 모델을 병행합니다. 자율 에이전트는 실행 전 입력을 차단(Input Gating)하고, 분석 파이프라인은 실행 후 출력을 검증(Output Validation)합니다.
