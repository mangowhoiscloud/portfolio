발견: ContextVar 스레드 미전파

Python ContextVar는 asyncio Task에는 자동 전파되지만 threading.Thread에는 전파되지 않습니다. CLIPoller가 daemon thread에서 실행되면서 AgenticLoop의 ContextVar를 읽지 못했습니다.

해결: copy_context()

스레드 생성 시 copy_context().run()으로 부모 컨텍스트를 명시적으로 복사합니다. 이 패턴은 SharedServices 전체에 적용되었습니다.

zombie job 정리

thin-only 전환 중 action 필드가 빈 스케줄러 잡 13개 발견. NL Parser가 생성했으나 action을 채우지 않은 잡입니다. 전수 삭제 후 create 시 action 필수 검증 추가.

ColorSystem 문제

serve가 /dev/null로 리다이렉트되면 Rich가 ColorSystem=None으로 판단. ANSI 코드 미출력. IPC 모드에서 ColorSystem을 강제 설정하여 해결.
