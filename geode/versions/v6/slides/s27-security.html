Bash 3-Layer 실행 제어

L1 — Syntax Validation

rm -rf / · chmod 777 · curl | bash 패턴 차단. 실행 전 정적 분석.

L2 — PolicyChain Hook

46 Hook Events 중 BASH_EXEC_PRE에서 PolicyChain 통과 여부 확인. 거부 시 즉시 차단 + 사유 반환.

L3 — macOS Seatbelt Sandbox

REODE: 34-패턴 deny-list. 파일시스템 쓰기, 네트워크 외부 접근, 프로세스 fork 제한.

Dual Trust Model: 두 독립 축

에이전트 안전과 파이프라인 검증은 독립 축이다. 에이전트는 입력을 차단(Input Gating), 파이프라인은 출력을 검증(Output Validation). 둘 다 통과해야만 결과가 신뢰된다.
