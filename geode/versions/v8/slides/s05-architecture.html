왜 6-Layer인가?

3-Layer(Presentation/Business/Data)는 에이전트에 맞지 않는다. 도구·메모리·오케스트레이션이 독립적으로 교체되어야 한다.

  • LLM 교체 → L4만 변경, 나머지 불변
  • 메모리 백엔드 교체 → L5만 변경
  • 도구 추가 → L3만 확장
