Trade-off: 무한 루프 위험 vs 자율성

while(tool_use)는 본질적으로 종료 보장이 없다. 5가지 종료 경로(자연 종료, 수렴 감지, 강제 텍스트, LLM 에러, 사용자 취소)로 탈출을 보장한다.

  • end_turn: LLM이 tool_use 없이 응답
  • max_rounds: 50라운드 도달
  • convergence: 동일 에러 3회 반복
  • force_text: 잔여 2라운드 강제
  • budget_stop: 비용 상한 초과

왜 max 50 rounds인가: 실측 기반

평균 4.2 라운드에 자연 종료(end_turn)한다. 복잡한 IP 분석에서도 30 라운드를 넘기지 않는다. 50은 예측 불가 상황의 안전망이지, 정상 실행 범위가 아니다.
