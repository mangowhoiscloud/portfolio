TRADE-OFF

always-on vs HITL — serve 모드는 HITL 없이 deny-by-default로 운영. DANGEROUS 등급 자동 차단

geode serve — Headless Daemon

$ geode serve --headless

Mode: HEADLESS

HITL: DISABLED

Policy: DENY_BY_DEFAULT

Interactive 모드 vs Serve 모드

Interactive

사용자 입력 대기. HITL 승인 요청. WRITE/DANGEROUS는 pause

Serve (Headless)

HITL 없음. SAFE/AUTO만 실행. WRITE 이상은 deny-by-default 차단

왜 deny-by-default인가

24/7 자율 실행 환경에서 사람이 검토 없이 WRITE 권한이 열리면 의도치 않은 파일/DB 수정이 발생. 최소 권한 원칙으로 안전한 자율성 확보
