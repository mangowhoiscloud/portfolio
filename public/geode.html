📈 CUSUM Change-Point Detection 📈 CUSUM Change-Point Detection

CUSUM(Cumulative Sum) 알고리즘을 사용하여 시장 트렌드의 변화 지점을 모니터링합니다. 누적 합계가 임계값을 초과하면 경고를 발생시킵니다. Monitors market trend change points using the CUSUM (Cumulative Sum) algorithm. Triggers alerts when cumulative sum exceeds thresholds.

CUSUM 공식 CUSUM Formula 

S_n = Σ(μ - x_i) / σ

# Where:
#   S_n = Cumulative sum at step n
#   μ  = Expected mean
#   x_i = Observed value at step i
#   σ  = Standard deviation

ClaudeClient 구현 ClaudeClient Implementation 

class ClaudeClient:
    def __init__(self, config: GeodeConfig):
        self.client = Anthropic(api_key=config.anthropic_api_key)
        self.model = config.model_name
        self.max_retries = config.max_retries

    async def analyze(
        self,
        system_prompt: str,
        user_prompt: str,
        response_model: Type[BaseModel],
    ) -> BaseModel:
        for attempt in range(self.max_retries):
            try:
                response = self.client.messages.create(
                    model=self.model,
                    system=system_prompt,
                    messages=[{"role": "user", "content": user_prompt}],
                )
                return response_model.model_validate_json(
                    response.content[0].text
                )
            except Exception:
                await asyncio.sleep(2 ** attempt)
        raise MaxRetriesExceeded()
